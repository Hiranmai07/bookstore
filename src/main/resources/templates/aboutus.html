About Us

The Resource Library for Educational Materials is a web-based application designed to manage and access educational resources. Built using Spring Boot, Thymeleaf, and MySQL, it allows users to add, view, and delete resources such as articles, tutorials, and documents. This system is ideal for organizing and sharing learning materials efficiently, providing a user-friendly interface and robust backend for data management.
